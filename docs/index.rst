Normandy Actions
================
Normandy is the server-side component of the `Recipe Server`_. It serves bundles
of JavaScript to various clients based on certain rules. The
`Normandy Actions repository`_ contains the implementations of actions that are
served by Normandy.

.. _Recipe Server: https://wiki.mozilla.org/Firefox/Recipe_Server
.. _Normandy Actions repository: https://github.com/mozilla/normandy-actions
